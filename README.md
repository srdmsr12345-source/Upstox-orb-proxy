# Ultimate NSE AI Scanner

AI-powered Indian Stock Market Scanner built with Python, Flask, Upstox API and NSE Data.

## Features

- Bottom Fishing Scanner
- Stage 2 Scanner
- Smart Money Scanner
- Volume Breakout Scanner
- Relative Strength Scanner
- ORB Scanner
- AI Ranking Engine
- NSE Bhavcopy Support
- NSE Delivery Data Support
- Upstox Live API
- Historical Data
- REST API
- Bootstrap Dashboard

## Technology

- Python 3.11
- Flask
- Pandas
- Requests
- Gunicorn

## Project Structure

```

app.py

requirements.txt
render.yaml

modules/

static/
data/

```

## Installation

```

pip install -r requirements.txt

python app.py

```
